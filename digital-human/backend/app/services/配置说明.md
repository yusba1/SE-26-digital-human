# dihuman_core.py 配置说明

## 配置参数位置

所有可配置参数都在文件头部（第 7-29 行），方便修改。

## 配置参数说明

### ONNX 模型文件配置

```python
UNET_MODEL_NAME = "unet.onnx"  # UNet 模型文件名
ENCODER_MODEL_NAME = "encoder.onnx"  # 音频编码器模型文件名
```

**说明**:
- `UNET_MODEL_NAME`: 你训练的数字人生成模型文件名
- `ENCODER_MODEL_NAME`: 音频编码器模型（通常不需要修改）

**修改示例**:
```python
UNET_MODEL_NAME = "tao_48_unet.onnx"  # 使用你转换的模型
```

### 数据文件夹配置

```python
IMG_INFERENCE_DIR = "img_inference"  # 图像文件夹名称
LMS_INFERENCE_DIR = "lms_inference"  # 关键点文件夹名称
```

**说明**:
- `IMG_INFERENCE_DIR`: 存储训练视频每一帧图像的文件夹
- `LMS_INFERENCE_DIR`: 存储每一帧面部关键点的文件夹

**修改示例**:
```python
IMG_INFERENCE_DIR = "my_img_inference"  # 使用自定义文件夹名
LMS_INFERENCE_DIR = "my_lms_inference"
```

### 帧数配置

```python
MAX_FRAMES = 300  # 最大加载帧数
```

**说明**:
- 控制最多加载多少帧图像
- 如果训练视频有 1000 帧，建议设置为 1000 或更大
- 帧数越多，内存占用越大

**修改示例**:
```python
MAX_FRAMES = 1000  # 加载 1000 帧
```

### 图像文件格式

```python
IMG_EXTENSION = ".jpg"  # 图像文件扩展名
LMS_EXTENSION = ".lms"  # 关键点文件扩展名
```

**说明**:
- 通常不需要修改
- 如果图像是 PNG 格式，可以改为 `.png`

### 音频处理配置

```python
AUDIO_SAMPLE_RATE = 16000  # 音频采样率 (Hz)
AUDIO_FRAME_SIZE = 160  # 每帧音频采样点数 (10ms at 16kHz)
```

**说明**:
- `AUDIO_SAMPLE_RATE`: 音频采样率，通常为 16000 Hz
- `AUDIO_FRAME_SIZE`: 每帧音频的采样点数（10ms = 160 samples at 16kHz）
- **不建议修改**，除非你了解音频处理细节

### 推理配置

```python
USE_GPU = True  # 是否使用 GPU 加速
```

**说明**:
- `True`: 优先使用 GPU（CUDA），如果不可用则回退到 CPU
- `False`: 只使用 CPU

## 使用示例

### 示例 1: 使用自定义模型

```python
# 修改模型文件名
UNET_MODEL_NAME = "tao_48_unet.onnx"
```

### 示例 2: 增加加载帧数

```python
# 如果你的训练视频有 500 帧
MAX_FRAMES = 500
```

### 示例 3: 使用自定义数据文件夹

```python
# 使用不同的文件夹名称
IMG_INFERENCE_DIR = "my_training_images"
LMS_INFERENCE_DIR = "my_training_landmarks"
```

## 修改后的生效方式

修改配置后，需要**重启后端服务**才能生效：

```bash
cd /root/digital-human
./stop.sh
./start.sh
```

## 注意事项

1. **模型文件名**: 确保模型文件存在于 `data_path` 目录下
2. **文件夹路径**: 确保图像和关键点文件夹存在于 `data_path` 目录下
3. **文件命名**: 图像和关键点文件必须从 0 开始连续编号（0.jpg, 1.jpg, ...）
4. **帧数限制**: `MAX_FRAMES` 不要设置过大，避免内存溢出

## 完整配置示例

```python
# ==================== 配置参数（修改这里即可）====================
# ONNX 模型文件配置
UNET_MODEL_NAME = "tao_48_unet.onnx"  # 使用你训练的模型
ENCODER_MODEL_NAME = "encoder.onnx"  # 保持不变

# 数据文件夹配置
IMG_INFERENCE_DIR = "img_inference"  # 训练图像文件夹
LMS_INFERENCE_DIR = "lms_inference"  # 关键点文件夹

# 帧数配置
MAX_FRAMES = 500  # 根据实际帧数设置

# 图像文件格式
IMG_EXTENSION = ".jpg"  # 图像格式
LMS_EXTENSION = ".lms"  # 关键点格式

# 音频处理配置（通常不需要修改）
AUDIO_SAMPLE_RATE = 16000
AUDIO_FRAME_SIZE = 160

# 推理配置
USE_GPU = True  # 使用 GPU 加速
# ================================================================
```
